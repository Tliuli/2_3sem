*(5Б) Перегрузка операторов. Операторы по умолчанию.
*Операторы можно определять. Делается для новых типов данных, по сути - перегрузка существующих операторов. 

Ключевое слово operator объявляет функцию, указывающую, что означает оператор-символ при применении к экземплярам класса. Это дает оператору более одного значения — "перегружает" его. Компилятор различает разные значения оператора, проверяя типы его операндов.

*(5Б) Инкапсуляция - что такое.
Инкапсуляция может быть рассмотрена как:
а) связь данных с методами которые этими данными управляют;
б) набор инструментов для управления доступом к данным или методам которые управляют этими данными.

*(5Б) Внешнее и внутреннее связывание.
- Если внутри единицы трансляции у объекта или функции есть внутреннее связывание, то этот символ виден компоновщику только внутри этой единицы трансляции. 
- Если же у объекта или функции есть внешнее связывание, то компоновщик сможет видеть его при обработке других единиц трансляции. Использование ключевого слова static в глобальном пространстве имен дает символу внутреннее связывание. Ключевое слово extern дает внешнее связывание.

Компилятор по умолчанию дает символам следующие связывания:
а) Non-const глобальные переменные — внешнее связывание;
б) Const глобальные переменные — внутреннее связывание;
в) Функции — внешнее связывание.

*(5Б) Ключевое слово const. Константные переменные, указатели, функции, методы.
- для переменной: создает переменную, значение которой изменить нельзя;
- для указателя: действие модификатора может распространяться либо на значение указателя, либо на данные на которые указывает указатель
- для функций: слово const используется по тем же правилам, что при описании обычных данных
- дл методов: определяют, что можно сделать с классом без побочного действия на его состояние, и что изменяет его состояние.
