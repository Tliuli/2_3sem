(5Б)  Что значат слова о с++: компилируемый, статически типизированный, высокого уровня.
Компилируемый - т.е. что происходит преобразование исходного кода на языке C++ в машинный код с помощью компилятора языка С++;
Статически типизированный - т.е. если для переменной был определен тип данных, то в последующем он не изменяем;
Высокого уровня - т.е. язык не зависит от внутреннего машинного кода. Более компактный и понятный для программиста, чем низкоуровневый. 

(5Б)  Перечислите основные фундаментальные типы данных и что в них следует хранить.
int - для целых чисел - 39;
double - для чисел с плавающей точкой - 3.5;
char - для символов - '~';
string - для строк - "abc~";
bool - для логических переменных - true.

(5Б)  Диапазоны значений фундаментальных типов данных, занимаемое место.
int - 4 байта (32 бита); [−2 147 483 648; 2 147 483 647];
double - 8 байт (8 бит); 1,7E +/- 308 (15 знаков);
char - 1 байт (8 бит); диапазон зависит от конкретной реализации, либо [-128; 127] - по умолчанию, либо [0; 255];
string - разные строки занимают разное кол-во памяти;
bool - 1 байт (8 бит); false или true/

(5Б)  Что такое определение, объявление.
Определение - объявление, выделяющее память для объекта;
Объявление - инструкция, приписывающая объекту определенное имя;
Объект - участок памяти, в котором хранится значение определенного типа.

(5Б)  Что такое операторы. Перечислите арифметические операторы для целых типов.
Оператор - это команда, обозначающая определенное математическое или логическое действие, выполняемое с данными;
...для целых типов: присваивание =, сложение +, вычитание -, умножение *, деление /.
